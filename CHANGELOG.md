# Changelog

本文件记录 cc-pet 面向用户的主要变更。版本号遵循
[Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.0.0] — 2026-07-15

这是 cc-pet 的首个公开版本。

### 新增

- 提供原创黄色小猪的 7 组纯矢量 Lottie 状态动画。
- 通过 Claude Code Hooks 和本地 Unix Domain Socket 感知会话状态。
- 支持多会话管理、系统通知、桌面气泡和可拖拽悬浮窗。
- 支持从设置面板安装和检查 Hook。
- 提供 Swift 测试、公开信息扫描以及 macOS 应用和 DMG 打包脚本。
